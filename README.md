# 🎯 Java Quiz Application

A console-based Quiz Application built using Java that simulates a real-time quiz game experience with lifelines and reward system.

## 🚀 Features

- ✅ 10 questions with increasing difficulty
- 💰 Reward-based system
- 🎮 Lifelines:
  - 50-50
  - Audience Poll
  - Skip Question
- ⌨️ User interaction using console (Scanner)
- 🧩 Modular code structure using methods

## 🛠️ Technologies Used

- Java
- OOP Concepts
- Scanner Class (java.util)

## 📌 How It Works

1. User starts the quiz
2. Each correct answer increases reward
3. Lifelines can be used when needed
4. Wrong answer ends the game
5. Safe points secure minimum winnings

## 🧠 Learning Outcomes

- Strong understanding of Java basics (loops, conditions, methods)
- Improved logical thinking and problem-solving
- Experience in building flow-based applications
- Handling user input effectively
